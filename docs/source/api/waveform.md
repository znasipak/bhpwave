# Waveform Module

The complex waveform $h = h_{+} - i h_{\times}$ is calculated as a sum of many harmonics,
\begin{align}
h(t, r, \theta, \phi) = \frac{\mu}{r} \sum_{lm} H_{lm}(t) e^{i \Phi_{lm}(t)} {}_{-2} Y_{lm}(\theta, \phi)
\end{align}
